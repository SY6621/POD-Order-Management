## ETSY 订单系统端到端回归测试用例

### 一、用途说明

本用例表用于每次上线前，对「待确认订单」页面相关的端到端流程进行回归测试：
从 **新订单 → 效果图生成 → 邮件生成与确认 → 待创建订单检查**，并覆盖四个 Tab 的展示与主要按钮。
