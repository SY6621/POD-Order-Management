Skills补充方案

创建 .qoder/skills/业务背景快速加载.md

# 业务背景快速加载 ## 何时触发 当用户说"需要业务背景"、"新功能开发"、"不了解项目"时自动加载 ## 项目核心信息 ### 系统目标 ETSY平台订单流自动化后台管理系统，实现从订单获取到发货的全流程自动化 ### 6大任务流程 1. 订单获取：抓取QQ邮件 → 清洗数据 → 提取订单信息 2. 生成效果图：反推产品编号 → 调取模版 → 生成JPG/SVG → 发送确认 3. 获取物流面单：API对接物流系统 → 获取单号和面单 4. 生成生产文档：整合信息 → 生成A4打印PDF 5. 物流状态查询：8天后查询 → 处理异常 6. 追评：客户收货后发送追评邮件 ### 技术栈 - 前端：Vue 3 + Vite + Element Plus + Pinia - 后端：Python + Poetry + ReportLab - 数据库：Supabase ### 当前进度 详见用户提供的【当前状态】部分 ## 下一步操作建议 根据用户提供的【下一步任务】，先读取相关文档，再执行开发

新对话标准流程

步骤1

用户操作

1 打开新窗口（D:\ETSY_Order_Automation）
2 复制通用开场白
3 填写【下一步任务】
4 根据任务类型勾选附加文件
5 粘贴发送
步骤2

AI响应流程

1 读取附加文件（按勾选顺序）
2 理解当前状态和下一步任务
3 如需业务背景，自动读取项目概述.md
4 确认开发规范
5 开始执行任务
